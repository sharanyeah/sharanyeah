# Sharanya - Observation Studio

## Overview
A personal portfolio website for Sharanya - a creative portfolio showcasing work, skills, and contact information. Built with plain HTML, CSS, and JavaScript.

## Project Structure
- `index.html` - Main HTML file with all page sections
- `styles.css` - All styles for the site
- `script.js` - JavaScript for interactivity (doodle canvas, form, animations)

## Features
- Responsive design with mobile navigation
- Interactive doodle canvas with brush/eraser/highlighter tools
- Contact form with doodle attachment
- Scroll observer with random observations
- Text highlight detector
- Enhanced marginalia mode (press M to toggle)
- Logo easter egg (click the dot 4 times)
- Smooth scroll navigation

## Tech Stack
- HTML5
- CSS3 (custom properties, flexbox, grid, animations)
- Vanilla JavaScript (ES6+)
- Google Fonts (Instrument Serif, Caveat, Space Mono)

## Development
Run the dev server:
```bash
npm run dev
```

The site is served on port 5000.

## Deployment
Configured for static deployment. All files are in the root directory.
